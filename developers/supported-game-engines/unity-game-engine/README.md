# Unity Game Engine

## Specific Requirements

### Supported Versions

* 2020.3+ (tested 2020.3)
* 2021.1+ (tested 2021.1)

### Supported Platforms

* Standalone Windows
* Standalone Windows 64
* WebGL
* Android
* iOS

### Supported Render Pipelines

* Built-in Render Pipeline

Future:

* High Definition Render Pipeline
* Universal Render Pipeline
